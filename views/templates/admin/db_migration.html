Instructions

  1. Modifiez le schéma XML de la base de données.
  2. Placez-vous dans le répertoire models/
  3. Exécutez propel diff. La commande doit générer un fichier models/build/migrations/PropelMigration_xxx.php.
  4. Exécutez ./migrate.sh. Le script doit générer un fichier migration.sql.
  5. Exécutez propel om pour mettre à jour les classes des modèles.
  6. Dans phpMyAdmin, ouvrez la fenêtre SQL, et sélectionnez le fichier migration.sql précédemment généré. Exécutez-le.
